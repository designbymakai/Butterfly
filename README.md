# Butterfly: ADHD Productivity App

Butterfly is a productivity app designed to help adults with ADHD manage tasks, calendars, and responsibilities. Built with React, TypeScript, and Electron, the app integrates an AI assistant to streamline task prioritization and keep users organized and focused.

## Features

- **Task System**: Organize tasks using projects and tags, view an at-a-glance dashboard, and manage tasks with icons and colors for easier differentiation.
- **AI Assistant**: Custom-trained AI helps prioritize tasks, create new ones, and manage your daily schedule.
- **Custom Calendar**: Create, customize, and manage events with drag-and-drop functionality and month-at-a-glance views.
